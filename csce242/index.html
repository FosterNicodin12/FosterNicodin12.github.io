CSCE 242: Website Applications

Nathan Nicodin

Students will utilize HTML and CSS to design the front-end of a website; matching a given specification. Secondly, they will use JavaScript to add functionality to their website. Lastly, they will learn to utilize node and server side JavaScript to store and retrieve data from a server.

Assignments

  • Assignment 1: Basic HTML
  • Assignment 2: Basic CSS
  • Assignment 3: Page Layout
